###### 1. var, let, and const
var is the old way, it ignores block scope and can cause bugs. 
let is the modern version and can be reassign. 
const is when the value wont change.

###### 2. Spread operator (...)
it just "spreads" things out.if you have an array and wanna copy it or merge two arrays — [...arr1, ...arr2]. works on objects too.

###### 3. map() vs filter() vs forEach()
map — loops and returns a new array (transformed stuff)
filter — loops and returns only items that pass a condition
forEach — just loops, doesnt return anything.

###### 4. Arrow functions
its just a shorter way to write functions. instead of function add(a,b) { return a+b } you write const add = (a,b) => a+b. also this behaves diffrently inside them.

###### 5. Template literals
instead of doing "Hello " + name + "!" we just write `Hello ${name}!` with backticks. we can also do multi-line strings.
